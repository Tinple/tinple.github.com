3.4

在例3.3 中我们编写了一个程序用以计算f(x,y)的值。这个函数的定义如下

f(x,y) =
x + y x ≥ 0 and y ≥ 0
x + y2 x ≥ 0 and y < 0
x2 + y x < 0 and y ≥ 0
x2 + y2 x < 0 and y < 0

在这里我们要求用if 的嵌套结构来编写这个程序。

x = input('Enter the x coefficient: ');
y = input('Enter the y coefficient: ');
if x >= 0 & y >= 0
   fun = x + y;
elseif x>=0 & y < 0
   fun = x + y^2;
elseif x < 0 & y >= 0
   fun = x^2 + y;
else
   fun = x^2 + y^2;
end

3.6

编写一个程序允许使用者输入一个字符串，这个字符必须是一个星期中的一天(即 “Sunday”,“Monday”,“Tuesday"等)，应用switch 结构把这些字符串转化为相应的数字，以星期天为第一天，以星期六为最后一天。如果输入不是这七个字符串中的一个，那么输出提示信息。

week = input('Please input a day in a week: ','s');
switch week
   case 'Sunday'
       sprintf('%s', 'Ò»ÐÇÆÚµÄµÚÒ»Ìì')
   case 'Monday'
       sprintf('%s', '一个星期的第一天')
   case 'Tuesday'
       sprintf('%s', '一个星期的第二天')
   case 'Wednesday'
       sprintf('%s', '一个星期的第三天')
   case 'Thursday'
       sprintf('%s', '一个星期的第四天')
   case 'Friday'
       sprintf('%s', '一个星期的第五天')
   case 'Saturday'
       sprintf('%s', '一个星期的第六天')
   otherwise
       sprintf('ERROR!')
end
