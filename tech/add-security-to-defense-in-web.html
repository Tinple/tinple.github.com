Add security to defense in web

2013-10-10 by Tinple

It would be a hard decision for me to write down this essay.But I still want to express some security opinion and give the solution for some attack that I used to do.

Mysql privileges

Like some other man, I used Mysql as my database to store datas in my last project. But I am not professional, just use it to combine with php. I didn't care the privileges when I executed a sql statement, just let it runned as I expected.

And now I realized the big potential problem that I have made. I didn't set privileges 精确地 for mysql users.

Think about an online shopping platform which includes manage page for admin to easily 操作 something he wants.If we don't care about privileges and just use the 默认的 settings, the problem is the 顾客 has same privileges as admin.

So we needs to 区分 the privileges between 顾客 and admin.
